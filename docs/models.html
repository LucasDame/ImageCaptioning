Encoder CNN Lite — Architecture couche par couche

Maths — opérations clés
Convolution 2D
output pixel (i, j) y[i,j] = Σm,n x[i·s+m, j·s+n] · W[m,n] + b
s=stride, W=filtre appris, b=biais
Batch Normalization
normalise chaque channel sur le batch BN(x) = γ · (x − μB) / √(σ²B + ε) + β
μ,σ² = moyenne/variance du mini-batch · γ,β = appris
MaxPool 2×2
garde le maximum dans chaque fenêtre 2×2 y[i,j] = max(x[2i, 2j], x[2i+1, 2j], x[2i, 2j+1], x[2i+1, 2j+1])
AdaptiveAvgPool → flatten
compresse (B, 256, 7, 7) → (B, 256) → (B, 512) f = Wfc · avg_pool(x) + bfc ∈ ℝ512
Pourquoi ces choix d'architecture ?
Stride 2 au lieu de MaxPool : le bloc 1 utilise stride=2 directement dans la Conv, ce qui réduit les paramètres.

Kaiming init : les poids sont initialisés avec W ~ N(0, √(2/fan_in)) pour que le signal ne s'écrase pas (ni explose) à travers les ReLU.

Dropout(0.5) : éteint 50% des neurones aléatoirement → force le réseau à ne pas sur-spécialiser.
Decoder LSTM — Architecture
Le LSTM reçoit un mot à la fois, et produit la distribution de probabilité sur le vocabulaire pour le mot suivant.
features (B, 512) → W_proj → h₀ (B, 512) = init LSTM Linear(512, 512)
↓ teacher forcing (entraînement)
captions (B, T) → Embedding(vocab, 256) → (B, T, 256)
LSTM(input=256, hidden=512) → (B, T, 512)
Dropout(0.5)
Linear(512 → vocab_size) → logits (B, T, V)
Softmax → P(mot suivant) ∈ [0,1]V
LSTM — Les 4 portes en détail
🚪 Oubli
ft = σ(Wf·[ht-1,xt] + bf)
0 = tout oublier / 1 = tout garder
📥 Entrée
it = σ(Wi·[ht-1,xt] + bi)
c̃t = tanh(Wc·[ht-1,xt] + bc)
💾 Cellule
ct = ft ⊙ ct-1 + it ⊙ c̃t
⊙ = produit élément par élément
📤 Sortie
ot = σ(Wo·[ht-1,xt] + bo)
ht = ot ⊙ tanh(ct)
Loss d'entraînement — Cross-Entropy L = − (1/T) · Σt=1T log P(wt | w<t, I)
I=image, w_t=mot correct au pas t

Bloc Résiduel (ResidualBlock) — le cœur de l'amélioration
L'idée clé : apprendre la résiduelle F(x) = H(x) − x plutôt que H(x) directement. La connexion "shortcut" garantit que le gradient remonte même dans les premières couches.
Chemin principal (residual)
Conv2d(in_ch → out_ch, k=3, stride=s) bias=False (BN s'en charge)
BatchNorm2d(out_ch)
ReLU
Conv2d(out_ch → out_ch, k=3, stride=1)
BatchNorm2d(out_ch)
Shortcut : x → Conv1×1 si dim≠ (sinon Identity)
Équation fondamentale du bloc résiduel y = ReLU( F(x, {Wi}) + shortcut(x) )

Si F(x) ≈ 0 → le bloc apprend l'identité (robustesse)
Gradient : ∂L/∂x = ∂L/∂y · (∂F/∂x + 1) → le "+1" évite le gradient nul

Beam Search — Algorithme complet (implémenté dans DecoderLSTM)
Au lieu de prendre le meilleur mot à chaque pas (greedy), le beam search garde les k meilleures hypothèses complètes. Complexité : O(k · V · T) au lieu de O(V · T).
Score d'une hypothèse (log-probabilité cumulée) score(w1:t) = Σi=1t log P(wi | w1:i-1, I)

Score final normalisé par la longueur score_norm = score / max(len, 1)
→ évite de favoriser les séquences courtes
Algorithme (beam_width = k) beams = [( score=0, tokens=[<START>], h )]

for t in range(max_length):
  for (score, tokens, h) in beams:
    logp = log_softmax(fc(LSTM(token[-1], h)))
    new_beams += top-k de logp
  beams = sort(new_beams)[:k]

best = argmax(score_norm for completed)
Exemple — beam_width=3
t=0
⟨START⟩ 0.0
t=1

une -0.8
t=2

un chat -0.9

t=3

EncoderSpatial — Pas de pooling global !
Même architecture résiduelle que EncoderCNN, mais on garde la dimension spatiale 7×7. Chaque cellule de la grille = une région de l'image.

Mécanisme d'Attention de Bahdanau
À chaque pas t, l'attention calcule un score d'importance pour chacune des 49 régions, puis pondère les features.
Étape 1 — energy score (pertinence région i) ei,t = vT · tanh( Wenc·fi + Wdec·ht-1 )
f_i ∈ ℝ512 = features région i · h_t-1 = hidden state courant
Étape 2 — softmax → poids α (somment à 1) αi,t = softmax(ei,t) = exp(ei,t) / Σj exp(ej,t)
α ∈ ℝ49 · Σᵢ αᵢ = 1 · αᵢ ≥ 0
Étape 3 — vecteur de contexte (moyenne pondérée) ctxt = Σi=149 αi,t · fi ∈ ℝ512
Paramètres entraînables de l'attention
W_enc ∈ ℝ512×256 · W_dec ∈ ℝ512×256 · v ∈ ℝ256
→ 256×512 + 256×512 + 256 = 262 400 params
DecoderWithAttention — Pas de temps t
e_t = Embed(w_{t-1})
+
ctx_t = Attention(f, h_{t-1})
↓ concaténation → (B, emb_dim + feat_dim)
LSTMCell(concat, (h_{t-1}, c_{t-1})) → h_t, c_t
Linear(hidden_dim → vocab_size) → logits
Initialisation h₀, c₀ depuis la moyenne des features h₀ = tanh(Wh · mean(f, dim=1))
c₀ = tanh(Wc · mean(f, dim=1))
mean(f) = (1/49) Σᵢ fᵢ — résumé global de l'image
